# Welcome to RewardApp
Spring Boot Reward Application for Customer : A retailer offers a rewards to its customers, awarding points based on each recorded purchase.
A customer receives 2 points for every doller over $100 i each transaction, plus 1 point for every doller spent between $50 and $100 in each  transaction.
RESTFul Register, Login, Logout and Transaction API.

Component :
Java - 17
Spring boot - 3.2.5
Spring-web
Spring-data-jpa
spring-security
spring-test
h2 database
